Instruções:

  1. Insira os dados dos clientes no campo abaixo, um cliente por linha.
  2. Os dados devem estar no formato: Nome do Cliente, Valor, Data de Pagamento (AAAA-MM-DD), Data de Renovação (AAAA-MM-DD)
  3. Exemplo: João Silva, 100, 2023-06-15, 2023-07-15
  4. Você pode copiar os dados do Excel ou de um arquivo de texto.
  5. Clique em "Visualizar" para verificar os dados antes de adicionar.
